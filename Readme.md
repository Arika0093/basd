#Bms Append Score Downloader(BASD)

##なにこれ
* BMS差分をダウンロードして導入するまで自動で行うツールです。
	* 既にある曲の差分のみ導入できます。
* 最初にセットアップさえしてしまえば、後はファイルのURLを指定するだけです。

##使い方
1. releasedから最新版のbasdをダウンロード、解凍しましょう。
2. basd.batをダブルクリックして起動します。
3. 初回起動時のみLR2のパスを求められます。指定してあげましょう。
	* 例: D:\LR2beta3\
4. 導入したい差分のURLを指定します。
	* 例1: http://hogehoge/bms.zip (差分置き場のURLでは無理です。本体のURLを指定してください)
	* 例2: up4010.rar(高難易度汎用アップローダーのものはupxxxxの形式で指定できます)
	* 例3: up2745(拡張子がzipのものについては省略することができます)
5. うまく行けば導入できます。失敗した場合はscoreフォルダに残骸が残るので各自処理してください。

##動作環境
* **.NET Framework 4.5**以上が必要です。つまり**Windows8以上**です。
* **64bit環境**でのみ動作します。今どき32bitの方が少ないですが。
